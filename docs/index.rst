Continuous Degradation Detection (CODED)
========================================

This is the documentation for the Continuous Degradation Detection (CODED) algorithm. CODED is open soure and can run on the `Google Earth Engine`_. CODED can be used to stratify a landscape based on areas of forest conversion and degradation. 

.. _Google Earth Engine: https://earthengine.google.com/
